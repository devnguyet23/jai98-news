Mở file `.env` trong thư mục gốc của project.

## Bước 2: Tìm và xóa các dòng cũ (nếu có)

Tìm và xóa các dòng sau (nếu có):
```
FIREBASE_ADMIN_PROJECT_ID=...
FIREBASE_ADMIN_CLIENT_EMAIL=...
FIREBASE_ADMIN_PRIVATE_KEY=...
```

## Bước 4: Lưu file `.env`

Lưu file và đóng lại.

## Bước 5: Restart Dev Server

```bash
# Dừng server hiện tại
lsof -ti:3000 | xargs kill -9

# Khởi động lại
npm run dev
```

## Bước 6: Test

```bash
npm run test:api:simple
```

Nếu thành công, bạn sẽ thấy API trả về dữ liệu thay vì lỗi!
